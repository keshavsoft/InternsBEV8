STEP 2 — Install Required Extensions

  • express api generator
  • express api sub routes generate
  • express api endpoints generate

After installing all extensions, restart VS Code once.

What We Actually Did Using Extensions

  • Installed all required KeshavSoft VS Code extensions
  • Generated Express boilerplate using ks-express-boilerplate

  • Created Express API route hierarchy without manually writing routing boilerplate
